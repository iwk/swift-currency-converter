Currency Converter App

Project Information

  1. Project was created with Xcode Version 7.0.1 and Swift 2.1 (lastest versions).

  2. Input number is formatted in realtime while user is editing the text field. It has to handle some special cases especially trailing character cases. Details are writtern in test cases and comments.

  3. To make the custom control(swiping currency control) reusable, colors are configurable in Interface Builder with IBInspectable. It uses IBDesignable to do live rendering in storyboard.

  4. The input text field is cleared when user starts inputing numbers. Original number is restored if user cancels editing.

  5. Checked Helvetica is one of the preinstalled fonts that come with iPhone/iPad since version 3/4. It should be ok.

  6. It includes a project reference folder containing the original instructions in case the online version changes.
